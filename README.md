# Customer Churn Prediction

## 📌 Problem Statement
Customer churn is a major issue affecting revenue. This project aims to predict which customers are likely to leave.

## 🎯 Objective
Build a predictive model to identify at-risk customers and provide actionable insights.

## 📊 Dataset
- Source: [Kaggle / Company Data]
- Features: Customer demographics, usage patterns, subscription details

## 🧠 Approach
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building (Logistic Regression, Random Forest)
- Model Evaluation

## 📈 Results
- Achieved 85% accuracy
- Identified top churn factors:
  - Contract type
  - Monthly charges
  - Customer tenure

## 🛠️ Tools & Technologies
Python, Pandas, Scikit-learn, Matplotlib

## 💡 Key Insights
- Long-term contracts reduce churn significantly
- High monthly charges increase churn probability

## 🚀 Future Improvements
- Deploy model using Flask / FastAPI
- Improve accuracy with deep learning
